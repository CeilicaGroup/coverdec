08Reglas de distribución y asignación de tareas
Principios generales
① Horas reales del Excel siempre — usar Hr_Pend. No inventar ni estimar.
② Secuencia de procesos obligatoria — no pintar sin lijar antes, no lijar sin ensamblar antes.
③ Días tope — imprimación máx. miércoles, pintura máx. jueves.
④ 0 horas extras como objetivo — redistribuir si alguien se sobrecarga.
⑤ Una tarea puede partirse entre días si supera la jornada.
⑥ Prioridad de proyectos — asignar huecos libres según orden 1→7.
⑦ No inventar procesos — si una lámpara no tiene CNC en el Excel, no asignar CNC.
⑧ John no aparece en tareas — solo en firma de órdenes de producción.

Lógica de distribución horaria
ALGORITMO SLOTS # Horario: 08:00–14:00 (slot 0→6) + pausa + 15:00–17:00 (slot 6→8) def slot_to_hhmm(slot): if slot >= 6: # tarde real = 15 + (slot - 6) else: # mañana real = 8 + slot return f"{int(real):02d}:{int((real%1)*60):02d}" # Ejemplo: slot 3.94 → 11:56 · slot 6.0 → 15:00 · slot 7.78 → 16:47 # Tarea que supera la jornada: continúa al día siguiente desde slot 0
